# Catálogo de Filmes
Um catálogo de filmes simples desenvolvido com HTML, CSS (Bootstrap) e JavaScript puro.

## Funcionalidades

- Listagem de filmes populares
- Busca por título de filmes
- Filtro por gênero
- Ordenação por popularidade, avaliação ou data
- Sistema de favoritos com localStorage
- Página de detalhes dos filmes

## Como usar

1. Registre-se no [The Movie Database](https://www.themoviedb.org/) para obter uma chave API gratuita
2. Substitua `SUA_CHAVE_API_AQUI` nos arquivos HTML pela sua chave da API
3. Abra o arquivo `index.html` em um navegador

## Estrutura de arquivos

- `index.html` - Página principal com lista de filmes
- `detalhes.html` - Página de detalhes de um filme
- `favoritos.html` - Página com filmes favoritos

## Tecnologias utilizadas

- HTML5
- CSS3 com Bootstrap 5
- JavaScript (ES6+)
- API The Movie Database (TMDb)
